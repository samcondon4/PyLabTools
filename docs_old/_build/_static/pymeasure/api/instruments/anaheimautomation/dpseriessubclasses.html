DP-Series Subclasses¶

This module implements subclasses of the DPSeriesMotorController class to implement additional functionality that falls out of scope of the pymeasure baseclass.

class pymeasure.instruments.anaheimautomation.LinearStageController(resource_name, log_id, homedir, **kwargs)¶

Bases: pymeasure.instruments.anaheimautomation.dpseriesmotorcontroller.DPSeriesMotorController

Represents a Anaheim DP Series Motor Controller driving a linear stage. This class overrides the home()

method and implements unit conversions between controller steps and distance along the linear stage.

Variables

  • homedir – Direction that the controller should turn the motor in a homing operation. Valid values are “CW” and “CCW” (default = “CCW”)

  • homespeed – Speed that the controller will turn the motor in a homing operation. Default = 500

  • turns_per_step – motor turns per individual controller step.

  • units_per_turn – distance in specified units that the linear stage travels in a single turn of the motor.
