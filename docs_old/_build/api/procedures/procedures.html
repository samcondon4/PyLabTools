class spherexlabtools.procedures.BaseProcedure(cfg, exp, hw=None, viewers=None, recorders=None, update_params=True, **kwargs)¶

Bases: spherexlabtools.thread.StoppableReusableThread

Thread to execute a measurement. Largely based on the pymeasure procedure class.

emit(record_name, record_data, inst_params=None, data_ts=None, filepath=None, **kwargs)¶

Post a record to the appropriate queues.

Parameters

  • record_name – String name of the record to post data to.

  • record_data – Data to be posted to the record queue.

  • inst_params – Optional set of instrument parameters to record.

  • data_ts – Timestamp of when the individual data point was generated.

  • filepath – String filepath to where the record should be saved if it is connected to a recorder. If set to None, the record.filepath is not modified as it may have been set by a controller.

  • kwargs – Key-word arguments that are passed to the handle method of the corresponding QueueThread.
