Wrapper Classes¶

Compound Instrument¶

class spherexlabtools.instruments.CompoundInstrument(cfg)¶

Bases: object

Dynamic class that merges two or more Instruments into a single object. CompoundInstruments are instantiated with a Compound Instrument Dictionary. See LINK TO TUTORIAL for an example of creating and using CompoundInstrument instances in a procedure.

merge_subinst_clsattrs(subinst, name, recursive=False)¶

Merge all subinstrument class attributes into the instance’s namespace.

Param

subinst: Instrument object to get properties from.

Param

name: String identifying the subinstrument.

Param

recursive: Recursively merge all subinstrument properties including those within baseclasses until the base Instrument class is reached.

shutdown()¶

Shutdown all subinstruments
