.. module:: spherexlabtools.recorders

spherexlabtools.recorders
##########################

| Specific recorder modules are currently implemented in three files:

    1. plaintext.py
    2. binary.py
    3. database.py

| Note that all recorders are imported into the spherexlabtools.recorders namespace so the import statement for any given recorder is simply:

.. code-block:: python

    from spherexlabtools.recorders import NameOfSpecificRecorder
